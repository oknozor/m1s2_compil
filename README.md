# Compil TP 1

### Dependencies
- rustc 1.34.0-nightly
- cargo
- gcc 
- babylon
- GNU indent (optional)


### build

```cargo build --release```

### run

The binary file is located in `target/release`, run it like so : ```./rjsc ${flag} ${args}```
If you are lost :`rjsc --help`.
